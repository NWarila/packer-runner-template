# Documentation

Documentation for this repository follows the [Diátaxis framework](https://diataxis.fr/) per [org ADR-0002](decision-records/org/0002-adopt-diataxis-documentation-framework.md) once that mirror lands in a subsequent PR.

| Quadrant     | Path                  | Purpose                              |
| ------------ | --------------------- | ------------------------------------ |
| Explanation  | `explanation/`        | Architecture, threat model, testing  |
| Reference    | `reference/`          | Mirroring, quality gates, protocol   |
| How-to       | `how-to/`             | Task-oriented guides                 |
| Decisions    | `decision-records/`   | ADRs (org-mirrored + repo-specific)  |

> The Diátaxis subdirectories and ADR mirrors land incrementally across P3–P7 of the phased build-out.
